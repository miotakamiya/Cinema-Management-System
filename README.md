Đề tài xây dựng Hệ thống Quản lý Rạp Chiếu Phim trên SQL Server nhằm số hóa toàn bộ quy trình vận hành của một rạp chiếu phim quy mô vừa, thay thế cách quản lý thủ công vốn dễ phát sinh sai sót. Hệ thống gồm hai thành phần: cơ sở dữ liệu xây dựng trên Microsoft SQL Server đạt chuẩn 3NF và ứng dụng giao diện viết bằng Python (CustomTkinter). Về nội dung, đề tài trải qua các bước từ phân tích yêu cầu, xây dựng mô hình thực thể liên kết (ER) với 10 thực thể, chuyển đổi sang mô hình quan hệ, chuẩn hóa dữ liệu đến triển khai truy vấn SQL với Stored Procedure và Trigger. Các chức năng chính bao gồm quản lý phim, phòng chiếu, loại ghế và mức phụ thu; đặt vé và hủy vé trong Transaction nguyên tử đảm bảo toàn vẹn dữ liệu; xem lịch chiếu và thống kê doanh thu theo từng bộ phim. Hệ thống sử dụng parameterized query phòng chống SQL Injection và được thiết kế theo kiến trúc Client–Server nội bộ, phù hợp triển khai thực tế tại rạp.
